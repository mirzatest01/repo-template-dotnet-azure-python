# Support

Thanks for using and contributing to our repositories.

---

## How to Get Help

### Bug Reports and Feature Requests
- Use GitHub Issues
- Select the appropriate issue template
- Provide sufficient context and details

---

### Questions and Discussions
For general questions:
- Use GitHub Discussions (if enabled), or
- Contact the repository maintainers

---

## What This Repository Does NOT Provide
Unless explicitly stated, repositories do not provide:
- 24/7 production support
- SLA-backed response times
- Real-time troubleshooting

---

## Internal Support (Employees/Partners)
If you are part of the organization:
- Follow internal support channels (Service Desk, Slack, Teams, etc.)
- Reference the repository name and issue/PR link where applicable

---

## Security Issues
Security issues must be reported following the instructions in `SECURITY.md`.
Do not raise them as public issues.
